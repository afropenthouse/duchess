# Vite React Shadcn TypeScript Project

A modern React application built with Vite, TypeScript, and Shadcn UI components.

## Deployment

### Vercel Deployment

1. Install Vercel CLI:
   ```bash
   npm i -g vercel
   ```

2. Deploy to Vercel:
   ```bash
   vercel
   ```

3. Follow the prompts to connect your project to Vercel

The project is already configured with a `vercel.json` file for optimal deployment settings.

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
